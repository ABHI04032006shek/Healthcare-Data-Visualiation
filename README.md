# 🏥 Hospital Data Analytics Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive **Hospital Analytics Dashboard** built using Power BI. It provides insights into hospital operations, patient data, and overall performance to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze patient trends and hospital activity
* Monitor key performance indicators (KPIs)
* Improve operational efficiency through data insights
* Enable interactive exploration of healthcare data

---

## 📊 Key Features

* 📈 Patient admission trends analysis
* 🧑‍⚕️ Department-wise performance
* 📅 Time-based filtering (daily/monthly/yearly)
* 🎛️ Interactive slicers and filters

---

## 🧰 Tools & Technologies

* **Power BI Desktop (.pbix)**
* Data Visualization
* Data Modeling (Relationships, DAX)
* Dashboard Design

---

## 📂 File Included

* `HOSPITAL_REPORT.pbix` → Main Power BI dashboard file

---

## 🚀 How to Use

1. Download the `.pbix` file
2. Open it in **Power BI Desktop**
3. Explore the dashboard using filters and slicers

---

## 💡 Insights Gained

* Identified peak patient admission periods
* Analyzed department performance variations
* Observed trends in hospital resource utilization
